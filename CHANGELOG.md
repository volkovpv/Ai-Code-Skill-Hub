# Changelog

Версии — SemVer. Первая запись файла всегда соответствует текущей версии в
`pyproject.toml` — это проверяет гейт `scripts/check_version_drift.py`.
Заголовок записи: `## [X.Y.Z] — YYYY-MM-DD`; текст записи становится
release notes на GitHub (его извлекает `.github/workflows/release.yml`).

## [0.0.0] — 2026-07-12

Базовая линия: опубликованных релизов ещё нет. Версия `0.0.0` никогда не
публикуется — релизы начинаются с первого поднятия версии.
